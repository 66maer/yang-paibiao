# 数据目录说明

此目录用于存储应用运行时产生的持久化数据。

## 目录结构

```
data/
├── bot/
│   └── screenshots/       # Bot 截图缓存
│       └── team_*.png     # 团队截图文件
```

## 截图缓存说明

- **位置**: `data/bot/screenshots/`
- **命名规则**: `team_{team_id}_{hash}.png`
  - `team_id`: 团队 ID
  - `hash`: 基于 (QQ群号 + 团队ID + 更新时间) 的 MD5 哈希值
- **缓存策略**:
  - 每个团队保留最近 10 个版本的截图
  - 当团队数据更新时，会生成新的缓存文件
  - 旧的缓存文件会被自动清理

## 注意事项

- 此目录中的所有内容都不会被提交到 Git（已在 `.gitignore` 中配置）
- 删除缓存文件是安全的，系统会在需要时自动重新生成
- 截图文件可能会占用一定的磁盘空间，可以定期手动清理旧文件
